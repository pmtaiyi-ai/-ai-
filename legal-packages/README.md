This folder contains 4 independent legal mini-sites (Privacy Policy + Terms of Service) generated from the user's Lumos templates.

Each site is intentionally styled and structured differently to avoid obvious visual/structural linkage.

Open the `index.html` in each app folder:
- `legal-packages/luneko/index.html`
- `legal-packages/tempio/index.html`
- `legal-packages/scentdiary/index.html`
- `legal-packages/bagcheck/index.html`
