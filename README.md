## GitHub Pages 部署

把本仓库发布到 GitHub 后，在仓库设置里开启 Pages：

1. 进入仓库 Settings。
2. 点击 Pages。
3. Source 选择 Deploy from a branch。
4. Branch 选择 `main`，目录选择 `/root`。
5. 保存后等待 GitHub 生成网址。

生成的网址通常是：

`https://你的用户名.github.io/仓库名/`
